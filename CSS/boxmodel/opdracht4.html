Boxmodel opdracht 4

Leg uit wat de volgende termen betekenen: content, padding, margin en border.

Content

Elk HTML element heeft een type content zoals, tekst, plaatje, video e.d. Een element kan een div zijn met daarin de content geplaatst.

Padding is de ruimte die je kunt geven aan de binnenkant van een element.

Margin is de afstand dieje kunt aangeven tussen de elementen.

Border is de rand van een element die je kunt stylen.
