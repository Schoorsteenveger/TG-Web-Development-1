Opdracht 1

Welke position values zijn er? Leg voor elke position values uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

Values

Position: static

Default waarde binnen html element. De properties top, right, bottom, left en z-index hebben geen effect.

Position: relative

Het element wordt gepositioneerd volgens de normale stroom van het document en wordt vervolgens ten opzichte van zichzelf verschoven op basis van de waarden boven, rechts, onder en links. De offset heeft geen invloed op de positie van andere elementen; dus de ruimte die voor het element in de paginalay-out wordt gegeven, is hetzelfde alsof positie statisch zou zijn.

Position: absolute

Het element wordt verwijderd uit de normale documentstroom en er wordt geen ruimte gemaakt voor het element in de paginalay-out. Het is gepositioneerd ten opzichte van zijn dichtstbijzijnde voorouder, indien aanwezig; anders wordt het relatief ten opzichte van het oorspronkelijke bevattende blok geplaatst. De uiteindelijke positie wordt bepaald door de waarden boven, rechts, onder en links.

Position: fixed

Een element met positie: vast; wordt relatief ten opzichte van het kijkvenster gepositioneerd, wat betekent dat het altijd op dezelfde plaats blijft, zelfs als de pagina wordt gescrold. De eigenschappen boven, rechts, onder en links worden gebruikt om het element te positioneren.

Position: sticky

Een element met positie: sticky; wordt gepositioneerd op basis van de scrollpositie van de gebruiker. Een plakkerig element schakelt tussen relatief en vast, afhankelijk van de schuifpositie. Het wordt relatief gepositioneerd totdat een bepaalde offsetpositie in de viewport wordt bereikt - dan "plakt" het op zijn plaats (zoals positie: vast).
