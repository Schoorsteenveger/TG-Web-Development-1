Pseudo elementen

14 verschillende pseudo elementen

Een CSS-pseudo-element is een trefwoord dat aan een selector is toegevoegd en waarmee u een specifiek deel van het/de geselecteerde element(en) kunt opmaken. Bijvoorbeeld

  1. ::before
  2. ::after
  3. ::first-line
  4. ::marker
  5. ::backdrop
  6. ::cue-region
  7. ::cue
  8. ::first-letter
  9. ::selection
  10. ::placeholder
  11. ::slotted()
  12. ::target-text
  13. ::grammar-error

Maak van de bovengenoemde pseudo elementen voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

::before (:before)

  • In CSS maakt ::before een pseudo-element dat het eerste onderliggende element is van het geselecteerde element. Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de eigenschap content. Het is standaard inline.

    • ::after (:after)

  • In CSS, ::after creëert een pseudo-element dat het laatste kind is van het geselecteerde element. Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de eigenschap content. Het is standaard inline.

    • ::marker

    Het ::marker CSS pseudo-element selecteert het markeringsvak van een lijstitem selecteert het markeringsvak van een lijstitem, dat doorgaans een opsommingsteken of nummer bevat. Het werkt op elk element of pseudo-element dat is ingesteld om weer te geven: lijstitem, zoals de (li).

    ::firstline

    Het ::first-line CSS pseudo-element past stijlen toe op de eerste regel van een block-level element. Merk op dat de lengte van de eerste regel van veel factoren afhangt, waaronder de breedte van het element, de breedte van het document en de lettergrootte van de tekst.

    ::backdrop

    The ::backdrop CSS pseudo-element is a box the size of the viewport which is rendered immediately beneath any element being presented in full-screen mode. This includes both elements which have been placed in full-screen mode using the Fullscreen API and elements. When multiple elements have been placed into full-screen mode, the backdrop is drawn immediately beneath the frontmost such element, and on top of the older full-screen elements.

    ::cue-region

    Het ::cue-regio CSS pseudo-element komt overeen met WebVTT-cues binnen een geselecteerd element. Dit kan worden gebruikt om bijschriften en andere aanwijzingen in media met VTT-tracks te stylen.

    De eigenschappen worden toegepast op de hele reeks cues alsof ze een enkele eenheid zijn. De enige uitzondering is dat de achtergrond en zijn longhand-eigenschappen van toepassing zijn op elke cue afzonderlijk, om te voorkomen dat er boxen worden gemaakt en onverwacht grote delen van de media worden verdoezeld.

    ::first-letter

    Het ::first-letter CSS-pseudo-element past stijlen toe op de eerste letter van de eerste regel van een element op blokniveau, maar alleen wanneer deze niet wordt voorafgegaan door andere inhoud (zoals afbeeldingen of inline-tabellen).

    ::selection

    Het ::selection CSS-pseudo-element past stijlen toe op het deel van een document dat door de gebruiker is gemarkeerd. Als je met de muis over de tekst in de pagina trekt, zie je de achtergrondkleur verschijnen.

    ::placeholder

    The ::placeholder CSS pseudo-element vertegenwoordigt de placeholder-tekst in een input of textarea element.

    Naam mist

    ::slotted()

    Het ::slotted() CSS-pseudo-element vertegenwoordigt elk element dat in een slot in een HTML-sjabloon is geplaatst (zie Sjablonen en slots gebruiken voor meer informatie). Dit werkt alleen bij gebruik in CSS die in een schaduw-DOM is geplaatst. Merk ook op dat deze selector geen tekstknooppunt selecteert dat in een slot is geplaatst; het richt zich alleen op daadwerkelijke elementen.

    ::target-text

    Het ::target-text CSS-pseudo-element vertegenwoordigt de tekst waarnaar is gescrolld als de browser scroll-naar-tekstfragmenten ondersteunt. Hiermee kunnen auteurs kiezen hoe ze dat gedeelte van de tekst willen markeren.

    ::grammar-error

    Het ::grammar-error CSS-pseudo-element vertegenwoordigt een tekstsegment dat door de user-agent is gemarkeerd als grammaticaal onjuist. Wellicht niet ondersteund door elke browser.
