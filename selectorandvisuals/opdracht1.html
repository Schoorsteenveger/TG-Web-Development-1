Selectors & Visuals

Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

  1. Inline

    2. Met behulp van een stijl attribuut in HTML-elementen.

    Binnen de HTML wordt styling toegevoegd met attributen als kleur, font familie, grootte van het font etc.

  2. Intern

    3. Met behulp van een style element in de head van het document.

    Boven de body plaats je een style tag met daartussen de css code. Elke div kin je een classe naam gevem om die dan te gebruiken in je externe stylesheet om de elementen op de pagina vorm te geven.

  3. Extern

    4. Met behulp van één of meer externe CSS-bestanden.

    Aparte folder of file met stlesheet en existentie CSS. De naam van je stylesheet schrijf je met link tag in de head van je HTML document.
