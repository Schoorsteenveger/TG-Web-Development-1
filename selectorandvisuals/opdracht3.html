Selectors & Visuals

Opdracht 3

Er zijn verschillende selectors, bijvoorbeeld;

  • Type selectors
  • Descendant selectors
  • Class selectors

Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

Type selectors

De naam van de selector is tegelijk op het type. Bijvoorbeeld body, a, p, h1, div...

Bijvoorbeeld de body in deze pagina heb ik een aparte stylesheet een achtergrondkleur gegeven. Voor de selector koos ik 'body'.


Descentdant selectors

Parent element met selector classe naam met daarnaast selectors geschreven die de kinderen voostellen, zelf ook weer elementen onder het parent element.

De elementen binnen een hoofdelement, bijvoorbeeld een div met class naam header, kunnen in één keer gestyled worden door de selectors naast elkaar te schrijven voor de accolade opent.


Class selectors

De naam van de selector bevat tegelijk de div met classenaam plus de p paragraaf die ik wil kleuren.

Een deel van de tekst heb ik gekleurd met een selecor die in 1 regel de div kiest met daarin het deel wat ik wil kleuren.
